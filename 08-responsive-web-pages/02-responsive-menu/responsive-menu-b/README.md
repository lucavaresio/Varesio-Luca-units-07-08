<h1>Responsive Menu a</h1>

For this exercise the website represent a typical page with simple section that contain a title
and a little paragraph

In the top of the page there's an header with a nav element that contain a list of link with
every fragment of the section

This exercise is for make this website responsive with the media queries using the max-width
propriety based on the width of the display

<hr />

<h2>How it works?</h2>

In the style.css resource, written with mobile-first method, there's the first rule
written for the display of the mobile phones (I'll take the example of my iPhone 13 display
that has a 470px about)

as you go up with the px width of the page the style changing and increases the height of the header,
the font-size of the single anchor in the nav menu
