<h1> Boxed In exercise </h1>

This exercise is a training for try and test a responsive web page based on
the width of the devices

<hr />

<h2> How it works </h2>

With the media queries written in the css code in the style.css source
take the min-width propriety based on the screen and for all the device with that viewport
that have a min-width of a specific bootstrap (like 568px for the mobile phone)
until to the next media queries with the next min-width propriety

For display that have viewport < 568px the image is not setted with the media query
but is written with normal css code

<h2> how to use it </h2>

With the dev mode in your browser, try the responsive mode and then try all the dimension
in pixel of the viewport

or try it on multiple devices
